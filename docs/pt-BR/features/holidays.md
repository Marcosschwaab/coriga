# Feriados

Gerencie feriados que afetam o preço das reservas.

## Funcionalidades

### Lista de Feriados
- Visualização em tabela de todos os feriados
- Mostra nome, data e tipo
- Filtre por ano

### Adicionar Feriado
- Clique em "Adicionar Feriado" para abrir o modal de criação
- Insira nome, data e tipo

### Tipos de Feriados
- **Nacional**: Feriados nacionais brasileiros
- **Municipal**: Feriados municipais
- **Comunidade**: Datas específicas do condomínio

### Editar/Excluir
- Clique no ícone de edição para modificar detalhes do feriado
- Clique no ícone de exclusão para remover o feriado

### Impacto nos Preços
- Feriados usam o preço de feriado da configuração de preços
- Exibidos com cor distinta no calendário
