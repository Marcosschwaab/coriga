# Reservas

Crie e gerencie reservas do salão de festas.

## Funcionalidades

### Lista de Reservas
- Visualização em tabela de todas as reservas
- Mostra data, morador, status, preço e status de pagamento
- Filtre por mês e status

### Criar Reserva
- Clique em "Nova Reserva" para abrir o modal de criação
- Selecione morador e data
- O preço é calculado automaticamente com base no tipo do dia

### Editar Reserva
- Clique no ícone de edição para modificar detalhes da reserva
- Atualize status ou altere a data

### Cancelar Reserva
- Clique em cancelar para marcar a reserva como cancelada
- A data fica disponível para novas reservas

### Exportação CSV
- Exporte reservas para arquivo CSV
- Inclui todos os detalhes da reserva
