# Painel (Dashboard)

O Painel fornece uma visão geral das estatísticas de reservas e eventos futuros.

## Funcionalidades

### Cards de Estatísticas
- **Total de Reservas**: Contagem de todas as reservas do mês selecionado
- **Confirmadas**: Número de reservas confirmadas
- **Canceladas**: Número de reservas canceladas
- **Receita**: Receita total das reservas

### Visão Geral de Receita
- Representação visual da receita mensal
- Comparação com meses anteriores

### Próximas Reservas
- Lista de reservas futuras
- Mostra nome do morador, data e status de pagamento
- Ordenadas por data (mais próxima primeiro)

## Navegação por Mês
- Use o seletor de mês para ver estatísticas de diferentes meses
- A visualização padrão mostra o mês atual
