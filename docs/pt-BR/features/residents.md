# Moradores

Gerencie os moradores do condomínio que podem fazer reservas do salão de festas.

## Funcionalidades

### Lista de Moradores
- Visualização em tabela de todos os moradores
- Mostra nome, unidade, telefone, email e status
- Colunas ordenáveis

### Busca
- Busque por nome ou número da unidade
- Filtragem em tempo real

### Adicionar Morador
- Clique em "Adicionar Morador" para abrir o modal de criação
- Campos obrigatórios: nome, unidade
- Campos opcionais: telefone, email

### Editar Morador
- Clique no ícone de edição para abrir o modal de edição
- Atualize qualquer informação do morador

### Gerenciamento de Status
- Badges de status Ativo/Inativo
- Desative o morador para impedir novas reservas
- A desativação preserva o histórico de reservas
