# Residents

Manage condominium residents who can make party room reservations.

## Features

### Resident List
- Table view of all residents
- Shows name, unit, phone, email, and status
- Sortable columns

### Search
- Search by name or unit number
- Real-time filtering

### Add Resident
- Click "Add Resident" to open creation modal
- Required fields: name, unit
- Optional fields: phone, email

### Edit Resident
- Click edit icon to open edit modal
- Update any resident information

### Status Management
- Active/Inactive status badges
- Deactivate resident to prevent new reservations
- Deactivation preserves reservation history
