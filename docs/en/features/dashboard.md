# Dashboard

The Dashboard provides an overview of reservation statistics and upcoming events.

## Features

### Statistics Cards
- **Total Reservations**: Count of all reservations for the selected month
- **Confirmed**: Number of confirmed reservations
- **Cancelled**: Number of cancelled reservations
- **Revenue**: Total revenue from reservations

### Revenue Overview
- Visual representation of monthly revenue
- Comparison with previous months

### Upcoming Reservations
- List of future reservations
- Shows resident name, date, and payment status
- Sorted by date (earliest first)

## Month Navigation
- Use the month selector to view statistics for different months
- Default view shows the current month
