# Pricing Configuration

Set and manage pricing for different day types.

## Features

### Price Fields
- **Weekday Price**: Price for Monday through Friday
- **Weekend Price**: Price for Saturday and Sunday
- **Holiday Price**: Price for holidays

### Save Configuration
- Click "Save" to update pricing
- New prices apply to future reservations
- Existing reservations keep their original price

### Pricing Rules
- Weekdays: Monday to Friday (non-holidays)
- Weekends: Saturday and Sunday (non-holidays)
- Holidays: Override weekend pricing
