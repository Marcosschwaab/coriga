# Holidays

Manage holidays that affect reservation pricing.

## Features

### Holiday List
- Table view of all holidays
- Shows name, date, and type
- Filter by year

### Add Holiday
- Click "Add Holiday" to open creation modal
- Enter name, date, and type

### Holiday Types
- **National**: Brazilian national holidays
- **Municipal**: City/municipal holidays
- **Community**: Condominium-specific dates

### Edit/Delete
- Click edit icon to modify holiday details
- Click delete icon to remove holiday

### Pricing Impact
- Holidays use the holiday price from pricing configuration
- Displayed with distinct color on calendar
