# Reservations

Create and manage party room reservations.

## Features

### Reservation List
- Table view of all reservations
- Shows date, resident, status, price, and payment status
- Filter by month and status

### Create Reservation
- Click "New Reservation" to open creation modal
- Select resident and date
- Price is automatically calculated based on day type

### Edit Reservation
- Click edit icon to modify reservation details
- Update status or change date

### Cancel Reservation
- Click cancel to mark reservation as cancelled
- Date becomes available for new reservations

### CSV Export
- Export reservations to CSV file
- Includes all reservation details
