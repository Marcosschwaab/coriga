# Calendar

The Calendar provides a monthly view of all reservations with visual indicators.

## Features

### Month View
- Grid layout showing all days of the month
- Previous/Next month navigation
- Current month highlighted

### Day Indicators
- **Reserved days**: Highlighted with reservation color
- **Holidays**: Marked with holiday indicator
- **Today**: Highlighted with current day border

### Color Coding
- Reservations are color-coded by status
- Payment status indicators on reserved days

### Status Filter
- Filter calendar view by reservation status
- Options: All, Confirmed, Cancelled

### Day Details
- Click on a day to view reservation details
- Modal shows resident info, payment status, and actions

### Legend
- Visual legend explaining color meanings
- Shows holiday indicators
